End-to-end, geometric competitive builder: DIA index + MS1 clusters + MS2 clusters (+ optional features) → candidates → geometric scoring/assignment → pseudo-MS/MS spectra.

Properties:

  • Each MS2 cluster participates in at most one precursor.
  • Physical program legality (group, tiles, apex-in-tile) is enforced inside PrecursorSearchIndex::enumerate_pairs(...).
