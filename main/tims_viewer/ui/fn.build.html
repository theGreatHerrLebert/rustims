Build the left control panel; mutates state and camera in place. tick_labels are the numeric axis-tick labels (rebuilt by the app on range change), vol_range lazily yields the volume density percentile pair; it is invoked ONLY when the user clicks the “Auto” transfer button, so the (multi-million-voxel) percentile scan never runs on the per-frame render path.
