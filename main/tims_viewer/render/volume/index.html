Module volume 

Source
Expand description

Volume rendering: a CPU max-intensity voxel grid (built incrementally as points stream in) uploaded to an anisotropic R16Float 3D texture, plus a fullscreen-triangle raycaster that shares the point cloud’s transfer-function + colormap.

MVP semantics: each voxel stores the MAX intensity of points falling in it (nearest voxel). Max keeps voxel values in the same range as raw per-point intensity, so the identical transfer function (i_min/i_max/log) and colormap apply unchanged — one shared exposure model across points and volume. Trilinear cloud-in-cell density deposition is the documented Phase-3 refinement.
